# Sorting Algorithm Visualizer 📊

A dynamic, interactive web application designed to visualize the step-by-step execution of fundamental sorting algorithms. 

**Live Demo:** [Click here to view the visualizer](https://lPrincel.github.io/Sorting-Visualizer/)

## Features
* **Real-Time Visualization:** Watch arrays get sorted in real-time through DOM manipulation.
* **Multiple Algorithms:** * Bubble Sort
  * Selection Sort
  * Insertion Sort
* **Randomized Data:** Generates a new, randomized array with a single click to test different scenarios.
* **Color-Coded Execution:** * 🟦 **Blue:** Unsorted elements
  * 🟥 **Red:** Elements currently being compared
  * 🟨 **Yellow:** Current minimum/key element
  * 🟩 **Green:** Fully sorted elements

## Tech Stack
* **HTML5:** Structuring the visual interface.
* **CSS3:** Flexbox for responsive bar alignment and styling.
* **JavaScript (ES6):** Core algorithmic logic, asynchronous functions (`async/await`, `Promises`), and DOM manipulation.

## How to Run Locally
1. Clone this repository to your local machine:
   `git clone https://github.com/lPrincel/Sorting-Visualizer.git`
2. Open the folder in your code editor.
3. Open `index.html` directly in your web browser. No server or backend installation required!
